Bem vindo a turma de Desenvolvimento Web Front-End da ECT/UFRN 2024.1

Nossa objetivo

Aqui você irá aprender uma variedade de tópicos essenciais para se destacar no desenvolvimento web moderno. Desde os fundamentos básicos de HTML, CSS e JavaScript até conceitos avançados de frameworks como React, este curso aborda tudo o que você precisa para construir aplicações web dinâmicas e interativas. Explore o poder do DOM, aprenda a criar e consumir Web Services em Node.js, e mergulhe em tópicos avançados como Event Loop, promises e async/await. Com uma abordagem prática e foco na aplicação real, você estará pronto para enfrentar desafios do mundo real no desenvolvimento frontend.
